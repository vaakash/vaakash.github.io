Shortcoder - PRO (v6.1)

Welcome to Shortcoder PRO plugin. Below new features, enhancements and bug fixes are made in this version.

  • New: Custom editors management page added to Shortcoder menu.
  • Fix: HTML is escaped in the editor sometimes.
  • Fix: Support for WordPress 6.1

6.0

  • New: Edit Shortcode content using block editor and page builder plugins like Elementor, WPBakery.
  • New: Revisions support for Shortcode content.
  • New: Include extra code to the footer when a Shortcode is used in a page.
  • New: Locate the usage of shortcode in posts and pages.
  • New: Prevent same shortcode nested loop.
  • New: New actions and filters introduced.
  • Fix: Post excerpt shortcode parameter now prints full post excerpt.
